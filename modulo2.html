Exemplos de Códigos:

Cálculo de Fatorial (usando loop):

def fatorial(n):
    resultado = 1
    for i in range(1, n + 1):
        resultado *= i
    return resultado

print(fatorial(5))  # Saída: 120

Verificação de Número Primo:

def eh_primo(numero):
    if numero < 2:
        return False
    for i in range(2, int(numero**0.5) + 1):
        if numero % i == 0:
            return False
    return True

print(eh_primo(17))  # Saída: True

Operações com Matrizes:

matriz = [[1, 2, 3],
          [4, 5, 6],
          [7, 8, 9]]

def transpor(matriz):
    return [[matriz[j][i] for j in range(len(matriz))] for i in range(len(matriz[0]))]

print(transpor(matriz))
