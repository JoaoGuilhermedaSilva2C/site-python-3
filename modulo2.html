Módulo II

Programas Básicos, Vetores, Matrizes e Arquivos

Detalhes do Módulo:

Neste módulo, você vai se aprofundar em conceitos fundamentais de programação e desenvolver habilidades para resolver problemas comuns com Python. Vamos abordar desde a lógica básica até a manipulação avançada de dados.

Além dos exemplos em aula, você terá acesso a exercícios práticos e projetos desafiadores que vão consolidar seu aprendizado. Cada tópico será complementado com exemplos de código bem comentados e explicações detalhadas.

O que você vai aprender neste módulo:

  • Programas Básicos:
    • Cálculo de Fatorial: Vamos explorar a lógica da multiplicação sequencial, abordando a implementação tanto com loops quanto com recursão. Esse conceito é fundamental em várias áreas da matemática e programação.
    • Verificação de Números Primos: Você vai aprender a identificar números primos utilizando uma lógica otimizada para reduzir o número de comparações, essencial em situações onde a eficiência é crucial.
    • Soma de Números: Iremos demonstrar como acumular valores em uma lista, abordando desde a soma simples até o uso de funções como `sum()` e loops personalizados para somas condicionais.
    • Sequência de Fibonacci: Descubra a lógica por trás dessa sequência clássica. Vamos explorar como gerá-la tanto usando loops quanto recursão, além de discutir aplicações práticas em análise de dados e algoritmos.
    • Algoritmos de Busca e Ordenação: Entenda como funciona a base dos algoritmos de busca (linear e binária) e de ordenação (Bubble Sort, Quick Sort), com exemplos de código comentados e comparações de desempenho.
  • Vetores e Matrizes:
    • Declaração e Inicialização de Vetores: Saiba como armazenar coleções de dados de forma eficiente em estruturas unidimensionais e aprenda a realizar operações comuns como acesso e alteração de valores.
    • Operações com Vetores: Vamos ensinar a realizar operações como soma de elementos, filtragem e busca. Além disso, você aprenderá a aplicar essas operações em situações reais, como processamento de listas de dados.
    • Trabalhando com Matrizes 2D e 3D: Iremos explorar a criação e manipulação de matrizes, abordando aplicações em gráficos, simulações e análise de dados. Você entenderá como organizar informações complexas em tabelas multidimensionais.
    • Multiplicação e Transposição de Matrizes: Aprenda as operações matemáticas mais comuns em matrizes e veja como aplicá-las em contextos como transformações geométricas, álgebra linear e análise numérica.
  • Manipulação de Arquivos:
    • Criar, Abrir e Fechar Arquivos: Inicie com as operações básicas de leitura e escrita em arquivos de texto, essenciais para projetos que envolvem armazenamento de dados persistente.
    • Leitura e Escrita em Arquivos de Texto: Entenda como manipular grandes volumes de dados, realizar leituras linha a linha ou carregar arquivos inteiros, além de armazenar saídas de programas.
    • Apagar Arquivos e Pastas: Aprenda a gerenciar o sistema de arquivos diretamente com Python, desde excluir arquivos até limpar diretórios inteiros de maneira segura e eficiente.
    • Trabalhando com Arquivos CSV: Explore o uso de bibliotecas como `csv` e `pandas` para manipular arquivos de dados estruturados. Aprenda como ler, filtrar e modificar tabelas para aplicações em ciência de dados e automação de processos.

Exemplos de Códigos:

Cálculo de Fatorial (usando loop):

def fatorial(n):
    resultado = 1
    for i in range(1, n + 1):
        resultado *= i
    return resultado

print(fatorial(5))  # Saída: 120

Verificação de Número Primo:

def eh_primo(numero):
    if numero < 2:
        return False
    for i in range(2, int(numero**0.5) + 1):
        if numero % i == 0:
            return False
    return True

print(eh_primo(17))  # Saída: True

Operações com Matrizes:

matriz = [[1, 2, 3],
          [4, 5, 6],
          [7, 8, 9]]

def transpor(matriz):
    return [[matriz[j][i] for j in range(len(matriz))] for i in range(len(matriz[0]))]

print(transpor(matriz))

Manipulação de Arquivos (Escrita e Leitura):

# Escrever em um arquivo
with open('dados.txt', 'w') as arquivo:
    arquivo.write('Este é um exemplo de escrita em arquivo.\n')

# Ler de um arquivo
with open('dados.txt', 'r') as arquivo:
    conteudo = arquivo.read()
