Aprenda os fundamentos da linguagem Go

1. Introdução ao Go Lang

Go, também conhecida como Golang, é uma linguagem de programação open-source criada pela Google. Ela é conhecida por ser simples, eficiente e muito utilizada em sistemas distribuídos e aplicações com alta concorrência.

Vamos começar com os conceitos básicos e os primeiros passos na programação com Go.


// Primeiro programa em Go: Olá Mundo
package main

import "fmt"

func main() {
    fmt.Println("Olá, Mundo!")
}

Explicação: Este é o primeiro programa básico em Go, onde usamos a função fmt.Println() para imprimir "Olá, Mundo!" no terminal. A função main() é o ponto de entrada obrigatório em programas Go.

2. Programas Básicos

Fatorial


// Cálculo do fatorial em Go
package main

import "fmt"

func fatorial(n int) int {
    if n == 0 {
        return 1
    }
    return n * fatorial(n-1)
}

func main() {
    fmt.Println("Fatorial de 5 é:", fatorial(5))
}

Explicação: Esta função calcula o fatorial de um número de forma recursiva. Se o número for 0, retorna 1. Caso contrário, multiplica o número pelo fatorial do número anterior até chegar a 1.

Número Primo


// Verificar se um número é primo
package main

import "fmt"

func isPrimo(n int) bool {
    if n <= 1 {
        return false
    }
    for i := 2; i < n; i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}

func main() {
    fmt.Println("7 é primo?", isPrimo(7))
}

Explicação: Este código verifica se um número é primo. A função isPrimo() percorre os números de 2 até o número dado e retorna false se encontrar um divisor. Se não houver divisores, retorna true.

Vetores e Matrizes


// Uso de vetores em Go
package main

import "fmt"

func main() {
    var vetor = [5]int{1, 2, 3, 4, 5}
    fmt.Println("Vetor:", vetor)
}

Explicação: O código demonstra como declarar um vetor em Go e acessar seus elementos. Aqui, um vetor de 5 inteiros é declarado e seus valores são impressos usando fmt.Println().

Trabalhando com Arquivos


// Criar e ler arquivos
package main

import (
    "fmt"
    "os"
)

func main() {
    // Criar arquivo
    arquivo, err := os.Create("exemplo.txt")
    if err != nil {
        fmt.Println("Erro ao criar o arquivo:", err)
        return
    }
    defer arquivo.Close()

    // Escrever no arquivo
    _, err = arquivo.WriteString("Olá, Go Lang!")
    if err != nil {
        fmt.Println("Erro ao escrever no arquivo:", err)
        return
    }
    fmt.Println("Arquivo criado e texto gravado.")

    // Ler o arquivo
    dados, err := os.ReadFile("exemplo.txt")
    if err != nil {
        fmt.Println("Erro ao ler o arquivo:", err)
        return
    }
    fmt.Println("Conteúdo do arquivo:", string(dados))
}

Explicação: Este código demonstra como criar e ler arquivos em Go. O arquivo "exemplo.txt" é criado, uma string é gravada nele e depois lida. Funções como os.Create() e os.ReadFile() são usadas para manipulação de arquivos.

3. Algoritmos de Busca e Ordenação

Busca Binária


// Implementação da busca binária
package main

import "fmt"

func buscaBinaria(vetor []int, x int) int {
    esquerda, direita := 0, len(vetor)-1
    for esquerda <= direita {
        meio := (esquerda + direita) / 2
        if vetor[meio] == x {
            return meio
        } else if vetor[meio] < x {
            esquerda = meio + 1
        } else {
            direita = meio - 1
        }
    }
    return -1 // Não encontrado
}

func main() {
    vetor := []int{1, 3, 5, 7, 9, 11}
    resultado := buscaBinaria(vetor, 7)
    if resultado != -1 {
        fmt.Println("Elemento encontrado na posição:", resultado)
    } else {
        fmt.Println("Elemento não encontrado.")
    }
}

Explicação: A busca binária procura um elemento em uma lista ordenada, dividindo o intervalo de pesquisa pela metade a cada iteração. O código retorna a posição do elemento se encontrado ou -1 se não estiver presente.

Ordenação por Inserção


// Implementação de ordenação por inserção
package main

import "fmt"

func ordenacaoPorInsercao(vetor []int) {
    for i := 1; i < len(vetor); i++ {
        chave := vetor[i]
        j := i - 1
        for j >= 0 && vetor[j] > chave {
            vetor[j+1] = vetor[j]
            j = j - 1
        }
        vetor[j+1] = chave
    }
}

func main() {
    vetor := []int{29, 10, 14, 37, 14}
    ordenacaoPorInsercao(vetor)
    fmt.Println("Vetor ordenado:", vetor)
}

Explicação: A ordenação por inserção percorre o vetor e insere cada elemento na sua posição correta em uma sublista crescente. O código reorganiza os elementos diretamente no vetor, sem a necessidade de uma estrutura adicional.
