Submódulo 1: Introdução ao Go Lang

Go, também conhecida como Golang, é uma linguagem de programação open-source criada pela Google. Nesta seção, aprenderemos conceitos básicos de Go e daremos os primeiros passos com a linguagem.

Primeiro Programa - "Olá Mundo"


// Primeiro programa em Go: Olá Mundo
package main

import "fmt"

func main() {
    fmt.Println("Olá, Mundo!")
}

Explicação: Este é um exemplo básico de programa em Go. Usamos a função fmt.Println() para exibir a mensagem "Olá, Mundo!" no terminal.

Submódulo 2: Programas Básicos

Cálculo do Fatorial


// Cálculo do fatorial em Go
package main

import "fmt"

func fatorial(n int) int {
    if n == 0 {
        return 1
    }
    return n * fatorial(n-1)
}

func main() {
    fmt.Println("Fatorial de 5 é:", fatorial(5))
}

Explicação: Este programa calcula o fatorial de um número usando uma função recursiva.

Submódulo 3: Algoritmos de Busca e Ordenação

Busca Binária


// Implementação da busca binária
package main

import "fmt"

func buscaBinaria(vetor []int, x int) int {
    esquerda, direita := 0, len(vetor)-1
    for esquerda <= direita {
        meio := (esquerda + direita) / 2
        if vetor[meio] == x {
            return meio
        } else if vetor[meio] < x {
            esquerda = meio + 1
        } else {
            direita = meio - 1
        }
    }
    return -1 // Não encontrado
}

func main() {
    vetor := []int{1, 3, 5, 7, 9, 11}
    resultado := buscaBinaria(vetor, 7)
    if resultado != -1 {
        fmt.Println("Elemento encontrado na posição:", resultado)
    } else {
        fmt.Println("Elemento não encontrado.")
    }
}

Explicação: A função buscaBinaria() implementa a busca binária para localizar um número em uma lista ordenada.
