1. Introdução ao Go Lang

Go, também conhecida como Golang, é uma linguagem de programação open-source criada pela Google. Ela é conhecida por ser simples, eficiente e muito utilizada em sistemas distribuídos e aplicações com alta concorrência.

Vamos começar com os conceitos básicos e os primeiros passos na programação com Go.

2. Programas Básicos

Fatorial


// Cálculo do fatorial em Go
package main

import "fmt"

func fatorial(n int) int {
    if n == 0 {
        return 1
    }
    return n * fatorial(n-1)
}

func main() {
    fmt.Println("Fatorial de 5 é:", fatorial(5))
}

Número Primo


// Verificar se um número é primo
package main

import "fmt"

func isPrimo(n int) bool {
    if n <= 1 {
        return false
    }
    for i := 2; i < n; i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}

func main() {
    fmt.Println("7 é primo?", isPrimo(7))
}

Trabalhando com Arquivos


// Criar e ler arquivos
package main

import (
    "fmt"
    "os"
)

func main() {
    // Criar arquivo
    arquivo, err := os.Create("exemplo.txt")
    if err != nil {
        fmt.Println("Erro ao criar o arquivo:", err)
        return
    }
    defer arquivo.Close()

    // Escrever no arquivo
    _, err = arquivo.WriteString("Olá, Go Lang!")
    if err != nil {
        fmt.Println("Erro ao escrever no arquivo:", err)
        return
    }
    fmt.Println("Arquivo criado e texto gravado.")

    // Ler o arquivo
    dados, err := os.ReadFile("exemplo.txt")
    if err != nil {
        fmt.Println("Erro ao ler o arquivo:", err)
        return
    }
    fmt.Println("Conteúdo do arquivo:", string(dados))
}

3. Algoritmos de Busca e Ordenação

Busca Binária


// Implementação da busca binária
package main

import "fmt"

func buscaBinaria(vetor []int, x int) int {
    esquerda, direita := 0, len(vetor)-1
    for esquerda <= direita {
        meio := (esquerda + direita) / 2
        if vetor[meio] == x {
            return meio
        } else if vetor[meio] < x {
            esquerda = meio + 1
        } else {
            direita = meio - 1
        }
    }
    return -1 // Não encontrado
}

func main() {
    vetor := []int{1, 3, 5, 7, 9, 11}
    resultado := buscaBinaria(vetor, 7)
    if resultado != -1 {
        fmt.Println("Elemento encontrado na posição:", resultado)
    } else {
        fmt.Println("Elemento não encontrado.")
    }
}

Ordenação por Inserção


// Implementação de ordenação por inserção
package main

import "fmt"

func ordenacaoPorInsercao(vetor []int) {
    for i := 1; i < len(vetor); i++ {
        chave := vetor[i]
        j := i - 1
        for j >= 0 && vetor[j] > chave {
            vetor[j+1] = vetor[j]
            j = j - 1
        }
        vetor[j+1] = chave
    }
}

func main() {
    vetor := []int{29, 10, 14, 37, 14}
    ordenacaoPorInsercao(vetor)
    fmt.Println("Vetor ordenado:", vetor)
}

4. Explicação dos Códigos

1. Olá Mundo

Este é o primeiro programa básico em qualquer linguagem de programação. Em Go, usamos a função fmt.Println() para imprimir uma mensagem no terminal. A função main() é o ponto de entrada do programa.

2. Fatorial

O código calcula o fatorial de um número usando recursão. A função fatorial() chama a si mesma até que o número chegue a zero, onde o valor de retorno é 1. A multiplicação dos valores sucessivos retorna o resultado.

3. Número Primo

Este código verifica se um número é primo. Ele percorre todos os números menores que o número dado e verifica se algum deles o divide exatamente. Se nenhum número o dividir, ele é primo.

4. Vetores

Este código mostra como declarar e acessar vetores em Go. Um vetor é uma coleção de elementos de tamanho fixo, e podemos acessá-los usando seus índices.

5. Arquivos

O código cria e manipula arquivos. Usamos os.Create() para criar um arquivo e os.ReadFile() para ler o conteúdo de um arquivo. Essas funções retornam erros que precisam ser tratados.

6. Busca Binária

A busca binária encontra um elemento em uma lista ordenada, dividindo o intervalo de busca pela metade a cada iteração. Se o elemento do meio for o desejado, ele é retornado. Caso contrário, o intervalo de busca é reduzido até o valor ser encontrado ou não.

7. Ordenação por Inserção

A ordenação por inserção é um algoritmo que constrói uma lista ordenada, movendo elementos menores à esquerda, uma posição de cada vez. É útil para pequenas listas, mas ineficiente para listas maiores.
