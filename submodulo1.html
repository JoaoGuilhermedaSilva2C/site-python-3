Submódulo 1 - Introdução ao Go Lang

Primeiros passos com a linguagem Go

O que é Go Lang?

Go, também conhecida como Golang, é uma linguagem de programação open-source desenvolvida pelo Google. Criada para ser eficiente, simples e segura, ela é amplamente utilizada em sistemas distribuídos, servidores e APIs. Seu modelo de concorrência eficiente e coleta de lixo embutida a tornam ideal para aplicações escaláveis e de alta performance.

Por que usar Go Lang?

  • Alto Desempenho: Compila diretamente para código de máquina, proporcionando uma performance comparável à de C/C++.
  • Sintaxe Simples: Tem uma sintaxe direta e limpa, facilitando a escrita e manutenção de código.
  • Concorrência Embutida: Oferece suporte nativo para programas concorrentes usando goroutines e canais.
  • Biblioteca Padrão Robusta: Inclui uma vasta biblioteca padrão que cobre desde manipulação de arquivos até funções de rede e criptografia.
  • Compilação Rápida: Compila rapidamente, o que acelera o ciclo de desenvolvimento.

Primeiro Programa - "Olá, Mundo!"

Para começar, vamos criar um programa básico em Go que exibe a mensagem "Olá, Mundo!" no console. Esse exemplo ajuda a entender a estrutura fundamental de um programa em Go.


// Primeiro programa em Go: Olá Mundo
package main

import "fmt"

func main() {
    fmt.Println("Olá, Mundo!")
}

Explicação do Código:

Vamos entender cada parte do código:

  • package main: Define o pacote principal do programa. Todo programa Go deve ter um pacote principal para ser executado.
  • import "fmt": Importa o pacote fmt, que contém funções para entrada e saída. Vamos usar fmt.Println para exibir a mensagem.
  • func main(): A função main é o ponto de entrada obrigatório do programa. Sem ela, o programa não executa.
  • fmt.Println("Olá, Mundo!"): Exibe "Olá, Mundo!" no console. Println imprime a mensagem seguida de uma nova linha.

Características Únicas do Go

Go inclui várias características que o diferenciam de outras linguagens de programação:

  • Goroutines: As goroutines são uma maneira leve de executar funções de forma concorrente, usando pouca memória e com troca de contexto eficiente.
  • Canais: Permitem a comunicação entre goroutines, facilitando a troca de informações entre processos simultâneos.
  • Compilação Direta: Compila rapidamente para código de máquina, eliminando o tempo de execução.

Instalando Go Lang

Para começar a programar com Go, você pode instalar o compilador Go em seu sistema. Siga estas instruções:

  1. Visite o site oficial: https://golang.org/dl/
  2. Baixe o instalador adequado para seu sistema operacional (Windows, macOS ou Linux).
  3. Siga as instruções para concluir a instalação.
  4. Para verificar a instalação, abra o terminal e digite go version para ver a versão instalada.

Rodando o Programa

Para compilar e rodar seu primeiro programa em Go:

  1. Salve o código em um arquivo chamado ola_mundo.go.
  2. Abra o terminal, navegue até o diretório do arquivo e digite: go run ola_mundo.go
  3. O terminal exibirá a mensagem Olá, Mundo! se o programa foi executado com sucesso.
