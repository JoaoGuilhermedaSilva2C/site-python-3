Programa 1 - Cálculo do Fatorial

O fatorial de um número é o produto de todos os números inteiros positivos até ele. Por exemplo, o fatorial de 5 é 120 (5 * 4 * 3 * 2 * 1).


// Programa para calcular o fatorial de um número
package main

import "fmt"

func fatorial(n int) int {
    if n == 0 {
        return 1
    }
    return n * fatorial(n-1)
}

func main() {
    numero := 5
    fmt.Printf("O fatorial de %d é %d\n", numero, fatorial(numero))
}

Explicação do Código:

  • func fatorial(n int) int: Define uma função recursiva que calcula o fatorial de um número.
  • numero := 5: Define o número para o qual queremos calcular o fatorial.
  • fmt.Printf: Exibe o resultado formatado no console.

Programa 2 - Verificação de Número Primo

Um número primo é divisível apenas por 1 e por ele mesmo. Neste programa, verificamos se um número é primo.


// Programa para verificar se um número é primo
package main

import "fmt"

func ehPrimo(n int) bool {
    if n <= 1 {
        return false
    }
    for i := 2; i*i <= n; i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}

func main() {
    numero := 7
    if ehPrimo(numero) {
        fmt.Printf("%d é um número primo\n", numero)
    } else {
        fmt.Printf("%d não é um número primo\n", numero)
    }
}

Explicação do Código:

  • func ehPrimo(n int) bool: Função que retorna verdadeiro se o número for primo e falso caso contrário.
  • for i := 2; i*i <= n: Checa divisores até a raiz quadrada do número para otimizar o cálculo.

Programa 3 - Manipulação de Vetores

Vetores são listas de elementos do mesmo tipo. Aqui, vamos iterar sobre um vetor e calcular a soma dos elementos.


// Programa para calcular a soma dos elementos de um vetor
package main

import "fmt"

func main() {
    numeros := []int{1, 2, 3, 4, 5}
    soma := 0

    for _, numero := range numeros {
        soma += numero
    }

    fmt.Printf("A soma dos elementos é %d\n", soma)
}

Explicação do Código:

  • numeros := []int{1, 2, 3, 4, 5}: Declara um vetor de inteiros com 5 elementos.
  • for _, numero := range numeros: Itera sobre cada elemento do vetor e adiciona à soma total.

Programa 4 - Manipulação de Matrizes

Matrizes são listas de listas, permitindo armazenar dados em duas dimensões. Este programa exibe os elementos de uma matriz 2x2.


// Programa para exibir os elementos de uma matriz 2x2
package main

import "fmt"

func main() {
    matriz := [2][2]int{{1, 2}, {3, 4}}

    for i := 0; i < 2; i++ {
        for j := 0; j < 2; j++ {
            fmt.Printf("%d ", matriz[i][j])
        }
        fmt.Println()
    }
}

Explicação do Código:

  • matriz := [2][2]int{{1, 2}, {3, 4}}: Declara uma matriz 2x2 com valores inteiros.
  • for i := 0; i < 2; i++: Itera sobre as linhas e colunas da matriz para exibir os valores.
